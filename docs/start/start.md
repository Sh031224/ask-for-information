## 개요

본 프로젝트는 대구소프트웨어마이스터고등학교 학생들이 직접 정보를 제공하는 문서를 작성하여 학교에 대한 정보와 분야별로 기초 지식을 다듬거나 학습할 수 있도록 제작하였습니다.

> 기초적인 지식 뿐만 아니라 이때까지 겪었던 경험도 같이 전달함으로써 보다 신뢰성 있는 정보를 제공합니다.
